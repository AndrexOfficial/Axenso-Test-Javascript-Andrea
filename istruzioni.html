Test Axenso - Javascript Frontend Developer

Dato un oggetto rappresentante un albero costituito da un insieme di sezioni e letture e considerato che le letture possono essere solamente nodi figli.

Realizzare le seguenti funzioni:

  • Estrazione di tutte le sezioni
  • Estrazione di tutte le letture
  • Conta di tutte le letture
  • Conta di tutte le sezioni
  • Cercare tra i nodi specificando in output se si tratta di una sezione o una lettura

NB: Le funzioni possono essere componibili

Bonus

Comporre le funzioni realizzate per ottenere una ui funzionante in grado di rappresentare l'albero, e che permetta interazioni utente tra cui: cercare, navigare i nodi o altro a piacere.
E' necessario rispettare l'indentamento rispetto ai nodi dell'albero.
Non è vincolante l'utilizzo di Framework particolari, la scelta può ricadere su uno qualsiasi tra quelli conosciuti.
Per la ui è consentito l'utilizzo di librerie css esterne (Bootstrap, Material ecc...)

Non è consentito utilizzare librerie esterne (helper).

La soluzione deve essere sotto versioning (scelta libera Github ecc...) e il repository deve contenere un piccolo Readme con istruzioni utili allo svilupppo, descrizione e start del progetto.
